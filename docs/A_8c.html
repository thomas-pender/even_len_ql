Detailed Description

String properties:

  • Vanishing sum of elements.
  • Representatives of orbits under the action of D_N x {+1, i, -1, j}.
  • Orbit representatives must have (1) real-valued correlations, and (2) have bounded square norm of nonprincipal character sums (<= 2N+2).

Executable usage: ./A <length> 1> <out_file>

  • <length> is a positive even integer denoting the length of the strings to be generated.
  • The generated bracelets are written to stdout. This can be redirected to an output file <out_file> as shown above.
